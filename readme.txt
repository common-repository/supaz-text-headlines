== Description ==
Supaz Text Headlines - A simple text headlines plugin

Supaz Text Headlines is a simple to use text headline plugin that enables you to post your news in a simple text highlight fashion. It is best suited if you want a quick large block text content in your site. It is best if you want a headline section for your newspaper site, magazine sites, personal blogs, etc.

==Features==
* Easy configuration
* Supports 4 new boards; 1 large board and 3 small boards
* Supports 20+ awesome headline fonts
* Supports 2 layouts: horizontal and vertical layhout
* Custom color and border settings.

== Installation ==
1. Unzip supaz-text-headlines.zip
2. Upload all the files to the /wp-content/plugins/supaz-text-headlines
3. Activate the plugin through the \'Plugins\' menu in WordPress.
4. For customizing the plugin\'s settings, click on Supaz Text Headlines option menu in WordPress left admin menu.
